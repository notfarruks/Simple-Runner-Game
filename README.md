Run & Jump -- Pygame Zero Mini Game

This is a small runner game, created using Pygame Zero, random, and Rect
from pygame, where the player's goal is to avoid enemies and reach 20
points by collecting coins.

HOW TO RUN: 1. Enter pip install pgzero into terminal 2. Place all
images in /images and sound files in /sounds 3. Run: pgzrun main.py on
terminal

CONTROLS: LEFT/RIGHT ARROW -- move SPACE -- jump

MENU: Start Game, Sound Toggle (ON/OFF), Exit

LIBRARIES: - pgzrun (Pygame Zero) - random - Rect from pygame

ASSETS: Sprites: Kenney.nl Sounds: mixkit.co
